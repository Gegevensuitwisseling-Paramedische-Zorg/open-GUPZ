# Inleiding
Een goede beveiliging van via het dataplatform beschikaar gestelde gegevens is een belangrijke vereiste. De te nemen beveiligingsmaatregelen richten zich op:

- Voorkomen dat een persoon, systeem of applicatie zich voordoet als iets of iemand anders en daardoor onterecht toegang verkrijgt tot het platform en de data die het platform beschukbaar stelt (spoofing)
- Voorkomen dat data in rust of transit ongeautoriseerd wordt aangepast (tampering)
- Onweerlegbaarheid van toegang tot het platform en de data die het platform beschikbaar stelt (non-repudiation)
- Voorkomen van denial of service
